# Jatin Kumar – Portfolio Website

A fully interactive, single-page portfolio built with **HTML · CSS · Vanilla JS · Three.js**.

🔗 **Live Site:** [jkbytecrafter.github.io/PortfolioWebsite](https://jkbytecrafter.github.io/PortfolioWebsite/)

---

## ✨ Features

| Feature | Details |
|---|---|
| **3D Hero** | Animated torus-knot rendered with Three.js, mouse-parallax effect |
| **Live LeetCode Stats** | Real-time solved count (Easy / Medium / Hard) via multi-endpoint API chain |
| **Live Codeforces Stats** | Rating, rank, max-rating via official Codeforces REST API |
| **Live GitHub Stats** | Repos, Followers, Following, Stars via GitHub REST API |
| **Project Flip Cards** | 3D CSS flip cards with tilt-on-hover |
| **Awards Timeline** | Animated scroll-reveal timeline |
| **Custom Cursor** | Magnetic dot + ring cursor |
| **Fully Responsive** | Mobile-friendly layout |

## 🛠️ Tech Stack

- **Three.js** – 3D hero canvas
- **LeetCode Stats API** – `leetcode-stats-api.herokuapp.com` (primary) → `allorigins.win` CORS proxy → `alfa-leetcode-api.onrender.com` (fallback chain)
- **Codeforces API** – `codeforces.com/api`
- **GitHub REST API** – `api.github.com`

## 🚀 Run Locally

Just open `index.html` in any modern browser — no build step required.

```bash
# Clone the repo
git clone https://github.com/JKBYTEcrafter/PortfolioWebsite.git
cd PortfolioWebsite

# Open directly (or use Live Server in VS Code)
start index.html
```

---

## 📡 Deploying on GitHub Pages

1. Go to **Settings → Pages** in this repo
2. Source: **Deploy from branch → `main` → `/ (root)`**
3. Site goes live at `https://jkbytecrafter.github.io/PortfolioWebsite/`
